# GM-PHOTO-EDITOR

**GM-PHOTO-EDITOR** is a desktop photo editing software developed with **PySide6** and Python.  
It features layers, brush & eraser tools, and basic image editing capabilities.

---

## Features

- Open and save images (PNG, JPG)
- Multiple layers with add/delete/select functionality
- Brush and eraser with adjustable color and size
- Simple Undo/Redo (basic version)
- Fully built with PySide6, no HTML/CSS/JS
- Cross-platform support (Windows/Linux/Mac)
